From my understanding a control flow can be compared to a daily to do list where your flow statements represent tasks that need to be completed throughout the day i.e. eat breakfast, go to the gym, etc. A loop is basically taking the exact same to do list and repeating it the next day.

"Describe what the DOM is and an example of how you might interact with it?"

The DOM (Document Object Model) defines a standard for accessing and updating the content, structure and style of a document and is made up of 3 different parts: the Core DOM, XML DOM and the HTML DOM.
By manipulating the DOM, web developers can create applications that update the data of the page without refreshing it. They can also create other applications that are customisable by the user.

"Explain the difference between accessing data from arrays and objects?"

Accessing data from an array is essentially like pulling data from a list with one variable containing different values (like a list of names) while accessing data from an object is like pulling properties (like shape, colour, size) from different variables.

"Explain what functions are and why they are useful?"

Functions put simply are programs inside of programs that are useful for storing reusable code and keeping things organised within a program as they help group variables together.
